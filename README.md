# bd-python2

## Objetivo:

Criar CRUD com python em uma estrutura de pacotes.

-Create(criar)
-Read(Buscar/Pesquisar)
-Update(Atualizar)
-Delete(Excluir)

### Tecnologias:
 
-ORM: SQLALchemy
-Containerização: Docker
-Banco de dados: MySQL
-Orquestração dde containers: Docker compose
-Versionamento Git
